## 🖼️ 권장 이미지 포맷 및 크기

### 일반 이미지
- 형식: WebP (대체: JPEG/PNG)
- 최대 너비: 1920px
- 최적화: 70-85% 품질

### 로고/아이콘
- 형식: SVG (벡터) 또는 PNG (투명 배경)
- 크기: 다양한 크기 준비 (예: 32px, 64px, 128px, 256px)

### 히어로 이미지
- 데스크톱: 1920x1080px
- 모바일: 768x1024px
- 형식: WebP 또는 최적화된 JPEG

## 📝 네이밍 컨벤션

- 소문자 사용
- 공백 대신 하이픈(-) 사용
- 설명적인 이름 사용
- 예시: `hero-background.webp`, `partner-samsung.png`, `icon-menu.svg`
